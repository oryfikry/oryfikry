§ 03 — Profile

A note on craft.

I care about the parts that don't show up in a screenshot — type latency on a 3G phone, an SQL plan that doesn't crumble at 10× load, the API contract that ages well three teams later.

Most of my time goes into the seam where classical web engineering meets AI: deterministic systems handing off cleanly to non-deterministic ones, with the right observability and fallbacks so failures are graceful, not silent.
